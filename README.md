# nodejs-web-server
A very simple Node.js web server using the connect and serve-static npm modules.
To use this server stick it in the folder of your project (or above it) and execute
```
$ npm install connect serve-static
```

Now launch the server with:
```
$ node server.js
```

Your website will be available at http://localhost:8080/yourfile.html or http://localhost:8080/<sub_folder>yourfile.html.
